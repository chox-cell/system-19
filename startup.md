# SYSTEM-19 STARTUP — v0

## Mission
Build the default open-source enforcement layer for AI-generated software changes.

## Core Thesis
AI coding will scale faster than human review quality. The winning layer is not another coder agent; it is the trusted gate before merge.

## Why Now
- AI-generated PR volume is increasing
- Teams need safer review workflows
- GitHub is the natural enforcement surface
- Open-source distribution can accelerate developer adoption

## Product Shape
Start with a GitHub Action that reviews AI-generated PRs and returns APPROVE / REVISE / BLOCK.

## Company Shape
- Open-source core
- developer-first distribution
- future enterprise layers for governance, audit, and policy depth

## North Star
Become the standard review layer for AI-built code.
